# gmKriger: Versions

### V0.1.0
- Initial release. 

### V0.2.0
- First working version, implemented based on formulation described in Pretell et al. (2024). 

### V1.0.0
- Formulation implemented based on Pretell et al. (2026) to address epistemic uncertainty in GMM selection. 
- Leverages the Cython and Python implementations of [geostats](https://github.com/RPretellD/geostats) and [gmms](https://github.com/RPretellD/gmms). 
- Limited applications to PGA, PGV, Ia, and CAV only. 
- Other code changes to accommodate the above updates.
